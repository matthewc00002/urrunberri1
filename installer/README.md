# UrrunBerri OS — Installateur automatique
## Statut : En cours de développement — en attente de validation

Ce dossier contient les fichiers pour créer une clé USB d'installation automatique
de Debian 13 + UrrunBerri OS en une seule étape.

### Contenu
- preseed.cfg — fichier de configuration automatique Debian
- urrunberri_setup.sh — script de configuration avant installation
- BUILD.md — instructions pour reconstruire l'ISO
